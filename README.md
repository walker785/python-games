# Games Python

Jogos feitos em python desenvolvidos durante o curso de Python 3 part 1 e 2 da Alura Cursos Online.

Jogos desenvolvidos:

	* Forca
	* Adivinhação

Os Jogos são administrados através de um programa pai que serve como menu para o usuário.
